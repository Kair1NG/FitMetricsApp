# FitMetricsApp

A wellness app designed to track weight gain and loss.

**Features**
- BMI Calculator
- Progress Tracker
-  -  Wellness Check
-  -  BMI Report
-  -  Weekly Report
-  Recommended Workouts
-  -  Workout Libraries
